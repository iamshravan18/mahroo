# Mahroo Amina Shaikh Website

Private residency and citizenship planning website for Indian HNI families, founders, and investors.

## Routes

- `/` - landing page
- `/fit-check` - Private Mobility Fit Check preview
- `/results/europe`
- `/results/uae`
- `/results/caribbean`
- `/results/panama`
- `/results/strategic-hni`
- `/results/explorer`
- `/results/manual-review`
- `/consultation`
- `/privacy`
- `/image-gallery` - internal image QA gallery

## Assets

All generated website assets are stored in `public/images/`.

The image manifest is `public/images/image-manifest.json`.

## Development

```bash
npm install
npm run dev
npm run build
```
